Quelques commandes essentielles de Git qui dépannent...

La balise git --help permet d'afficher toutes les commandes.
La balise git status permet d'afficher le statut des fichiers.
La balise git config permet de configurer git.

git config --globalpermet de configurer l'user.email et l'user.name.

git config --list permet de vérifier la configuration.

La balise git add permet d'ajouter un fichier à la liste des commits il s'agit d'une étape intermédiaire.

git add -A et git add --allpermettent d'ajouter tous les fichiers suivis et non suivis à la liste ds commits .

La balise git commit -m réalise le commit avec l'écriture d'un message de validation.

git commit -m "" réalise le commit avec l'écriture d'un message de validation en écrivant directement le message de validation entre parenthèses.

git commit -a valide tous les fichiers modifiés.

La balise git log affiche les commits précédents.

git log --oneline affiche les commits précédents sur une seule ligne.

git log nom de fichier affiche les commits précédents concernant le fichier spécifié.

La balise git diff permet d'afficher les évolutions d'un fichier.
La balise git checkout permet de créer une branche.

La balise git checkout nom de la branche de destination permet de changer de branche.

La balise man permet d'afficher toutes les options de la commande (tapée après man).
